Consumet

129 Stars

typescript

Consumet is an organization that was created in collaboration with a friend of mine.
After collaborating on an application, AnimeLazerV3, we realized that one of our bottlenecks as frontend developers was acquiring the needed data to build the application.

The main method of acquiring that data was scrapping webpages and we come up with the idea of writing an api that scrapes data; and our ambition was to create an API that scrapes all entertainment related data, viz. books, movies, anime, etc., so frontend developers can focus on... the frontend.

The idea turned out to be successful and now has an active community on both discord and github.
